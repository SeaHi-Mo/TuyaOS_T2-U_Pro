# 名词解释

归零码：归零码是指其有电脉冲宽度t小于码元宽度T，即信号电压在一个码元终止时刻前总要回到零电平。通常，归零波形使用半占空码（在灯珠芯片中均使用非半占空码），从归零码波形可以直接提取定时信息，它是其他码型提取位同步信息时常采用的一种过渡波形。

# 基本功能

提供设备（灯带）注册机制，将应用层颜色数据转换为ws2812对应的线序，并将转换后的颜色数据进一步转换为spi码流数据，通过spi将数据发送出去。
